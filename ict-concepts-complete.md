# 🐺 DHEEB ICT Concepts - Complete Guide

## Top-Down Analysis (MMXM)

### الفريمات:

| الفريم | الاختصار | الوظيفة |
|--------|---------|---------|
| Weekly/Monthly | HTF | Bias (الاتجاه) |
| Daily | MTF | Zones + Structure |
| 4H/1H | ITF | Setup |
| M15/M5/M1 | LTF | Entry |

### MMXM Rule:

```
Setup على فريم كبير + Entry على فريم صغير

مثال:
- Setup: H1
- Entry: M5 أو M1
```

## Core ICT Concepts

### 1. Order Blocks (OB)

| النوع | الوصف |
|------|-------|
| Bullish OB | شمعة/شموع قبل حركة صاعدة قوية |
| Bearish OB | شمعة/شموع قبل حركة هابطة قوية |

**القاعدة:**
- OB = آخر شمعة قبل displacement قوي
- كلما كبر الـ body = أقوى OB

---

### 2. Fair Value Gap (FVG)

| النوع | الوصف |
|------|-------|
| Bullish FVG | gap up (c1.high < c3.low) |
| Bearish FVG | gap down (c1.low > c3.high) |

**القاعدة:**
- FVG = منطقة دعم/مقاومة قوية
- الدخول عند retest لـ FVG

---

### 3. Liquidity (السيولة)

| النوع | الاختصار | الوصف |
|------|---------|-------|
| Buy Side Liquidity | BSL |低于区间的低点 |
| Sell Side Liquidity | SSL | highs |
| Previous Day High | PDH |昨天高点 |
| Previous Day Low | PDL |昨天低点 |

**القاعدة:**
- Price ي swept السيولة قبل movement كبير

---

### 4. Market Structure

| المفهوم | الاختصار | الوصف |
|--------|---------|-------|
| Market Structure Shift | MSS | كسر الـ swing low/high |
| Change of Character | CHOCH | تحول في السلوك |
| Break of Structure | BoS | كسر الـ structure |

---

### 5. Breaker Blocks

```
OB يكسر → يرجع يختبر = Breaker Block
```

| الخطوة | الإجراء |
|-------|---------|
| 1 | Order Block يتكون |
| 2 | Price يكسر OB |
| 3 | OB يصير Zone جديد |
| 4 | Price يرجع يختبر ← Enter |

---

### 6. Equilibrium (نقطة التوازن)

```
Equilibrium = (High + Low) / 2
```

- Midpoint of range
- Zone of balance
- Support/Resistance
