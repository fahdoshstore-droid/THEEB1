# 📊 Chart Analysis Template

---

## Header
```
📈 [SYMBOL] - [TIMEFRAME]
Price: [PRICE] | Change: [CHANGE%]
```

---

## TRIL Analysis (4 Lines)

```
🔺 T: [Time Session]
🔻 R: [Raid/Liquidity]
🔲 I: [Imbalance/FVG]
📍 L: [Location/Levels]
```

## ICT Concepts (6 Lines)

```
📉 Trend: [BULLISH/BEARISH]
📊 Structure: [MSS/CHOCH]
🧱 OB: [Order Block Zone]
💧 Liquidity: [Swept/Target]
⚡ FVG: [Filled/Open]
📐 Fib: [Levels]
```

## Entry Plan (3 Lines)

```
🎯 Entry: [Zone]
```

## Confluence Score

```
Confluence: [X]% - [STRONG/MODERARY/WEAK]

```
